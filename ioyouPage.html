IOyou web application and wireframes
This website was created as an application to calculate and minimze the number of money transfers by consolidating debts within a network of people. Users are able to input new transactions, and then visualize the consolidated money transfers with a "web" graphic.

I created this website as part of the group project for a Web Programming class at Tufts. My primary responsibility was in organizing the layou of the website and working on the front-end component. As each team-member finished their part, I put everything together to make it a working application. We utilized Bootstrap, AngularJS, HTML5/CSS, MongoDB, Javascript, and Jquery.
