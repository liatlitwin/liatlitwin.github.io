ChatSnap Mobile Application Prototype and Mockup
As a project in Human Computer Interaction (COMP 171) I chose to create an iOS app called "ChatSnap". The main function of this application is to enable the user to take a screenshot of a long text message conversation (that spans longer than the screen space on the iPhone) and thus capture the conversation in its entirety. Additional features of the app allow the user to store all images in albums and upload them to external platforms.
